# Drawing App + MicroPAL Plotting Examples

## Fields

Call phi_poly_verbose which returns a json-nested json data with fields:

- p: polynomial order (usually 2)
- coord: coordinates of nodes
- tri: triangle pointers to coord node (each row represents a triangle)
- u: solution field

## Settings

URL: https://micro-pal-qkntutwgva-uc.a.run.app/palarchive/pal
Function: phi_poly_verbose

## Cases

- triangle
- pentagon

Each case consists of Drawing App JSON, returned output, and a PNG of the plot.
